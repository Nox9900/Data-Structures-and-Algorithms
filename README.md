# Data Structures and Algorithms (DSA) Practice Repository

This repository contains a collection of C++ solutions to common data structures and algorithms problems from LEETCODE. The purpose of the project is to practice and demonstrate proficiency in solving coding challenges across multiple problem categories such as arrays, strings, dynamic programming, graphs, trees, and more.

## Structure

The workspace is organized into folders based on algorithmic patterns and data structures. Each folder groups related problems, allowing for easy navigation and reference:

- **arrays/** – array-based problems like `firstMissingPositive.cpp`, `majorityElement.cpp`, and `moveZeroes.cpp`.
- **strings/** – string manipulation challenges such as `isSubsequence.cpp` and `reverseWords.cpp`.
- **dynamic programming/** – problems involving dynamic programming techniques.
- **backtracking/** – backtracking algorithms and combinatorial search problems.
- **graphs/** – graph traversal and pathfinding problems.
- **trees/** – tree-based algorithms and traversals.
- **and more** – other categories including greedy, hash table, linked list, sliding window, etc.

## Usage

Copy or refer to individual `.cpp` files when studying or preparing for technical interviews. Add new problems and solutions as you continue practicing.
